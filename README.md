This project analyses customer shopping behaviour to uncover insights related to purchasing patterns, subscription impact, discounts, and revenue performance.

The project combines Python (EDA & data cleaning), PostgreSQL (analytical queries), and a Power BI dashboard to deliver business-driven insights.

The main objective was to answer:

Do subscribed customers spend more?

Which products and categories generate the most revenue?

How do discounts influence purchasing?

What customer segments contribute most to revenue?
